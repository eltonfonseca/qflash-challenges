# Quasar Flash - Challenges

## Setup

Install all dependencies of project with command

```bash
$ bundle install
```

## Challenge #0

For run the code, enter on directory and run on console

```bash
$ ruby main.rb
```

For run tests

```bash
$ rspec spec/custom_collatz_spec.rb
```
